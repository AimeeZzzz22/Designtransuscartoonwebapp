Design a cartoon-style desktop web app called TransUs with the tagline “Translate conflict into understanding.” TransUs is an AI-powered relationship reflection tool for couples that helps users better understand each other’s emotions, intentions, and misunderstandings after an argument. The app should feel warm, emotionally safe, playful, and comforting, not clinical or corporate. The visual style should be cute, polished, and startup-like, with a soft pastel palette, rounded corners, bubbly panels, expressive icons, speech-bubble shapes, heart accents, cozy clouds, and friendly cartoon illustrations that symbolize empathy, listening, and communication. Create a full landing page plus a main product interface. Main app interface Create a main app screen that feels like the real working product. At the top, include: the TransUs logo and name a small subtitle like AI reflection tool for couples a profile/menu area on the top right The main layout should be a two-column or centered card-based dashboard with a friendly cartoon aesthetic.
